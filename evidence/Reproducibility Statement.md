# Reproducibility Statement

## Boundary

- Chain: Ethereum
- Protocol: Aave V3
- Observation block: `20,000,000`
- State method: archive-capable historical `eth_call`
- Public release: methodology, bounded results, evidence index, and hashes

## Reproducible Components

An independent reviewer with archive access can query the named Aave interfaces at the target block, compare account aggregates with reserve-level state, and inspect the cited governance and automation contracts. Machine-readable artifacts record the experiment outputs used by the paper.

## Controlled Components

Credentials, reusable private extraction procedures, commercial review methods, and execution logic are excluded. Their exclusion limits turnkey replication of the internal pipeline; it does not convert controlled evidence into public evidence.

## Integrity

The artifact manifest records SHA-256 hashes for the canonical manuscript, metadata, evidence package, final PDF, and review records. Any post-release change requires a new version and release record.
